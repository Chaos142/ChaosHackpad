# Chaos's Hackpad

### Features
- 128x32 OLED display
- EC11 rotary encoder
- 4 SK6812 RGB LEDs (backlighting)
- 7 MX-Style keys
- WASD and volume controls (firmware will be updated in the future)

### CAD Model:
Screws together using 4 M3 screws.   
Comes in two pieces - the base and the top.
