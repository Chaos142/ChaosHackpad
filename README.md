# Chaos's Hackpad

## Features
- 128x32 OLED display
- EC11 rotary encoder
- 4 SK6812 RGB LEDs
- 7 MX-Style keys

## CAD Model:
Screws together using 4 M3 screws.   
Comes in two pieces - the base and the top.
